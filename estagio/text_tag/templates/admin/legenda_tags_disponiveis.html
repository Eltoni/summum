É possível utilizar tags dentro do texto escrito. Essas tags são como variáveis que retornam dinamicamente um valor esperado ao serem processadas.

Exemplo de uso: Texto com tag inserida: "...Informado aqui uma data aleatória {{date_random_tag}}..." Texto com tag processado: "...Informado aqui uma data aleatória 01/01/2000..." Opções adicionais: Informando parâmetros às tags: "...Informado aqui uma data aleatória {{date_random_tag|[-1]}}..." Texto com tag processado(com parâmetro): "...Informado aqui uma data aleatória 31/12/1999..." Texto com tag processado(sem parâmetro): "...Informado aqui uma data aleatória 01/01/2000..." Informando filtros às tags: "...Informado aqui uma data aleatória {{date_random_tag|option_filter}}..." Texto com tag processado(com filtro): "...Informado aqui uma data aleatória 31/12/1999..." Texto com tag processado(sem filtro): "...Informado aqui uma data aleatória 01/01/2000..."
{% endverbatim %}
